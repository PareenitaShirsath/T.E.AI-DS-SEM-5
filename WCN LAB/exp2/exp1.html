Elements

HTML elements are the building blocks of HTML pages. They consist of tags, attributes, and content.

Attributes

HTML attributes provide additional information about elements. They are always included in the opening tag.

Head

The element contains meta-information about the HTML document, such as its title, linked CSS, and metadata.

Body

The element contains the content of an HTML document, such as text, images, links, and other media.

Hyperlink

The (anchor) tag defines a hyperlink, which is used to link from one page to another. The href attribute specifies the link's destination.

The img tag is used to embed images in HTML pages. The src attribute specifies the path to the image.

Frames

The iframe tag specifies an inline frame. An inline frame is used to embed another document within the current HTML document.

Forms

HTML forms are used to collect user input. The form element wraps input elements like text fields, radio buttons, and submit buttons.










Multimedia

HTML supports multimedia elements like audio and video through the audio and video tags.

Links

The link tag defines the relationship between a document and an external resource, most commonly used to link to stylesheets.
